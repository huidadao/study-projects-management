# Project: YouTube Learning Tracker

## What This Is

Track YouTube video learning progress with hierarchical category management and visual dashboard. Individual learner tracks self-study videos from YouTube organized by category tree.

## Core Value

Track YouTube video learning progress with hierarchical category management and visual dashboard.

## Technical Stack

- **Backend:** FastAPI + SQLite + SQLModel
- **Frontend:** React + Vite + TypeScript + Tailwind CSS + Zustand
